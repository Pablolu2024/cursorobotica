Introducción a la Programación para Robots

Conceptos básicos de programación

Algoritmos


Definición: Un algoritmo es una secuencia de pasos ordenados y finitos que resuelven un problema o realizan una tarea específica. En programación, un algoritmo se convierte en un conjunto de instrucciones que una computadora puede ejecutar.

Ejemplo:


Un algoritmo para hacer una tostada podría ser:


1. Tomar una rebanada de pan.
2. Colocar el pan en la tostadora.
3. Encender la tostadora.
4. Esperar a que el pan esté tostado.
5. Sacar el pan tostado.

Bucles (Loops)


Definición: Un bucle es una estructura de control que permite repetir un bloque de código varias veces, hasta que se cumpla una condición específica o durante un número determinado de veces.

Tipos de Bucles:
- Bucle `for`: Repite un bloque de código un número determinado de veces.
- Bucle `while`: Repite un bloque de código mientras una condición sea verdadera.
- Bucle `do-while`: Similar al `while`, pero la condición se evalúa después de ejecutar el bloque de código al menos una vez.

Ejemplo:

Supongamos que queremos imprimir los números del 1 al 5:

python

for i in range(1, 6):
print(i)

Condiciones (If-Else Statements)

Definición: Las condiciones son estructuras de control que permiten tomar decisiones en el código. Evalúan una expresión lógica y, en función de si es verdadera o falsa, ejecutan diferentes bloques de código.

Sintaxis:


- if: Ejecuta un bloque de código si la condición es verdadera.
- else: Ejecuta un bloque de código si la condición es falsa.
- elif (else if): Proporciona una condición adicional si la primera condición es falsa.

Ejemplo:

Determinar si un número es positivo, negativo o cero:

python

numero = 10

if numero > 0:
print("El número es positivo")
elif numero < 0:
print("El número es negativo")
else:
print("El número es cero")
